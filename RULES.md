# Maze Crawler Competition Rules

## 1. COMPETITION-SPECIFIC TERMS
- **Title:** Maze Crawler
- **Sponsor:** Google LLC
- **Website:** https://www.kaggle.com/competitions/maze-crawler
- **Data Access and Use:** Apache 2.0
- **Team Limit:** 5 members
- **Submission Limit:** 20 per day, 2 final submissions for judging.
- **Scoring:** Based on performance in an episode (Simulation competition).

## 2. COMPETITION-SPECIFIC RULES
- **External Data:** Allowed if "reasonably accessible to all" and of "minimal cost" (Reasonableness Standard).
- **AutoML:** Allowed with appropriate license.
- **Simulation Environment:** Uses Kaggle Environments.
- **No Ingress/Egress:** Submissions cannot use external info or send info out during evaluation.

## 3. WINNER'S OBLIGATIONS
- Prizes are not awarded for this competition.
- Detailed description of methodology and code repository link may be required.
