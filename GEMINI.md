# Maze Crawler - Project Instructions

## Overview
This project is an entry for the Kaggle Maze Crawler competition, a 1v1 strategy simulation.

## Core Mandates
- **Environment:** 20x20 scrolling grid.
- **Goal:** Survive longer than the opponent.
- **Units:** Factory, Scout, Worker, Miner.
- **Mechanics:** Fog of war, energy consumption, wall destruction/construction, resource harvesting.

## Architecture
- `src/`: Core logic for the bot.
- `notebooks/`: Exploration and local testing.
- `tests/`: Unit and integration tests.

## Workflow
1. **Research:** Understand the `kaggle-environments` API for Maze Crawler.
2. **Strategy:** Develop a survival-focused strategy (efficient resource gathering and movement).
3. **Execution:** Implement the bot logic.
4. **Validation:** Run local simulations against baseline bots.
